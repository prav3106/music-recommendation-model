# music-recommendation-model
Music recommender that gives you 5 similar songs to the one you need.

It uses data from the dataset called 'data.csv' which contains features such as danceability, tempo etc. The ipynb file has code to scale the data and uses only the necessary features to make it runnable on colab. It uses the KNN algorithm (again for the model to be runnable on colab) so it can be a little inaccurate.
